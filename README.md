# XENEON EDGE Todo 正式版

專門給 Corsair XENEON EDGE Web URL Widget 使用的待辦工具。

## 功能

- 時鐘與日期
- 自由新增待辦
- 直接點擊文字修改
- 完成勾選
- 個別刪除
- 清除所有已完成項目
- 拖曳排序
- 完成進度條
- 使用瀏覽器 LocalStorage 自動儲存
- 適配 1024 × 600 橫向顯示

## 上傳到 GitHub

將下列檔案上傳到 repository 根目錄：

- `index.html`
- `style.css`
- `script.js`

GitHub Pages 設定：

1. Repository → Settings → Pages
2. Source 選擇 `Deploy from a branch`
3. Branch 選擇 `main`
4. Folder 選擇 `/ (root)`
5. 儲存並等待部署

網站網址：

`https://phantom10221.github.io/xeneon-edge/`

## XENEON EDGE 設定

在 iCUE 的 Web URL Widget 中直接填入：

`https://phantom10221.github.io/xeneon-edge/`

不要使用 `<iframe>`，也不要加上 `file:///`。

## 更新後仍看到舊畫面

GitHub Pages 可能需要約 1–3 分鐘部署。部署完成後，在 Chrome 按 `Ctrl + F5` 強制重新整理。
